# WWE 2025 Watchlist Tracker

A web application to track your WWE watching progress throughout 2025.

## Features
- **Schedule Tracking**: View all Raw, SmackDown, and PLE events for 2025.
- **Progress Tracking**: Mark shows as watched and track your completion percentage.
- **Cloud Sync**: Sync your progress across devices using Firebase Authentication.
- **The Wrestling Oracle**: AI-powered chat to ask about the schedule and storylines.
- **Story So Far**: AI-generated recap of your watched events.

## Setup

### Prerequisites
- A Firebase project (for Auth and Firestore).
- A Google Gemini API Key (for AI features).

### Installation
1.  Clone the repository.
2.  Open `index.html`.
3.  Update the `firebaseConfig` object with your Firebase project details.
4.  Update the `apiKey` variable with your Gemini API key.

## Hosting
This project is designed to be hosted on GitHub Pages or any static site hosting service.
